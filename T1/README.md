
# Chat APP INE5418

Servidor e cliente de um chat Python com Sockets

## Rodar localmente

Chat

Vá para o diretório do projeto

```bash
  cd T1
```

Instalar as dependências

```bash
  pip3 install -r requirements.txt
```

Inicializar o servidor

```bash
  python3 server.py
```

Executar o chat de usuário

```bash
  python3 client.py
```
